Many2Many Tags Links

  • The "Many2Many Tag Links" module for Odoo is a feature-enhancing extension that addresses the limitation of non-clickable many-to-many tag widgets in the Odoo system. This module empowers users to interact with many-to-many tags more intuitively by enabling direct access to the respective form view when tags are clicked. This streamlines the process of managing and customizing tags Widgets, making Odoo more user-friendly and efficient. Thus, "Many2Many Tag Links" is a valuable Odoo module that overcomes the limitations of non-clickable many-to-many tag widgets.
Supported Editions
Community
Enterprise

"Many2Many Tags Links" module supports both editions of Odoo, Community and Enterprise as well.

Demo Video

Features

Clickable Many-to-Many Tags: With Many2Many Tag Links, users can click on many-to-many tags to access their respective form views. This allows for quick and easy navigation to view and edit the details associated with each tag.

Effortless Tag Customization: Users can edit and customize tags with ease. By opening the form view directly, they can modify tag details, add descriptions, update categories, and make any necessary changes, simplifying the process of tag management.

Increased Productivity: Users can work more efficiently and effectively with the ability to quickly access and modify tag information. This leads to improved productivity in managing tags, which are a crucial organizational tool in Odoo.

How to Install

  • Open your Odoo ERP System ▶ Apps ▶ Install/Activate this module.


  • Once you've installed the "Many2Many Tag Links" module, you can start using it in Odoo. Let's say you're using the employee app, and you pick any random employee. In that employee's profile, there's a field called "roles," which contains tags.


  • With this module, you can now click on any of those tags, and it will open up a detailed screen with more information about that specific tag.
